# 🐍 Snake Game (Qt Framework)

## 🎯 Overview
This project is a **classic Snake Game** built in **C++ using the Qt Framework**.  
It serves as the **final project for Programming 2**, demonstrating the use of **Object-Oriented Programming (OOP)** concepts and **GUI development** with Qt.

## 🧰 Technologies Used
- **C++ (C++17/20)**
- **Qt Framework (Qt Widgets / QMainWindow / QTimer / QKeyEvent)**
- **Qt Creator IDE**
- **Event-driven programming**

## 🕹️ Gameplay
- Use **arrow keys** (↑ ↓ ← →) to control the snake.  
- Each time the snake eats food, it **grows in length**.  
- The game ends when the snake **hits a wall** or **collides with itself**.  
- Displays the **current score** and a **Game Over** screen with restart/exit options.
